### 说明

- OC 版本：0.78；
- Wi-Fi，蓝牙，随航，隔空投送正常；
- 开关机，睡眠，唤醒正常(测试时间较短，仅供参考)；
  唤醒说明：平时使用误触键盘或鼠标总会无端唤醒，所以这里仅支持按下电源键唤醒；
- HDMI / DP 显示正常（单显示）；
- 音频输出正常（麦克风未测试）；
- USB 测试正常；

  USB 定制说明：由于 15 端口限制，有线网卡左侧的两个 USB3.0 端口仅支持 3.0；
